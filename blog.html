Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Sets the HTML content inside an element.
innerText : sets the text content inside an element.
textContent : sets the text content content inside an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript used to handle events efficiently by taking advantage of the event bubbling mechanism.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that is triggered on an element bubbles up from the target element to its ancestors in the DOM hierarchy.
